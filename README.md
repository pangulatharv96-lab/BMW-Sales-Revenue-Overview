# BMW-Sales-Revenue-Overview
Overview

This project presents an end-to-end Data Analysis and Visualization of BMW’s global sales performance using Excel. The dataset includes detailed information such as model type, year, region, color, fuel type, engine size, mileage, price, and sales volume.
A fully interactive Excel Dashboard was created to explore key metrics and reveal insights into BMW’s sales trends across regions and vehicle categories.
Project Objective

The main aim of this project is to:
	•	Analyze BMW’s sales performance across different models and years
	•	Identify top-performing BMW models
	•	Understand regional demand trends
	•	Examine pricing patterns and sales volume distribution
	•	Build an interactive and visually appealing dashboard for quick business insights
  Dashboard Features

The Excel dashboard includes:
	•	Total Vehicle Count
	•	Average Sales Volume
	•	Maximum Price & Top Selling Model
	•	Top 3 Best-Selling BMW Models
	•	Regional Sales Distribution
	•	Price vs Sales Volume Analysis
	•	Fuel Type Demand Analysis
	•	Color Preference Trends
	•	Interactive charts and slicers for dynamic insights
  Conclusion

This project demonstrates how Excel can be used to transform raw data into meaningful business insights. Through structured analysis and dashboarding, BMW’s sales performance, pricing strategy, and regional demand trends become easy to understand for business stakeholders.
